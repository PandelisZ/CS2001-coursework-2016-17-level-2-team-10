The Search Widget

The search widget allows the user to search for their favourite movie simply by typing it in in the search movies bar. A scroll function is also incorporated into this feature that also allows the go through the list of movies and choose one they enjoy.

The search view also has a feature in which it matches the characters entered by the user to movies held in the database. The characters entered will reduce the list of available movies which then allows the user to select their preferred movie.
